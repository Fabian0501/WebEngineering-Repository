The Spread Operator (...)


What is the Spread Operator ?

The JavaScript spread operator (...) allows us to quickly copy all or parts of an existing array into another array.
It works by expanding an iterable, such as an array, into individual elements.
This means that instead of manually iterating over the elements of an array and copying them one by one,
we can simply use the spread operator to copy the entire array or specific parts of it in a more concise and efficient manner.


const originalArray = [1, 2, 3, 4, 5];
const copiedArray = [...originalArray];
What is the Output ?
console.log(copiedArray);


const array1 = [1, 2, 3];
const array2 = [4, 5, 6];
What is the Output this time ?
const combinedArray = [...array1, ...array2];
