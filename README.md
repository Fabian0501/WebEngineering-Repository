# WE-Repository

## This project showcases all the tasks I completed in the "Web Engineering" course.

### Description and Functions of the Image:

#### Main Functions:

1. **Home Icon (red highlighted):**
   - Clicking on it takes the user to the landing page (homepage).

2. **Numbers in the Sidebar:**
   - Clicking on these numbers allows the user to display different elements. These elements are loaded via an iframe into the main display area.

3. **Tasks as Iframes and Tasks as refs Buttons:**
   - These buttons allow the user to switch between different display modes for tasks (as iframes or references).
   - **Comming soon**

#### Main Display Area:
- The area shows the content of the selected task, which is loaded via an iframe.
